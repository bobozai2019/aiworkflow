# 多平台Agent系统设计方案

## 一、项目概述

构建一个基于Python的多平台智能Agent系统，支持国产大模型协议、多Agent协作编排，并具备移动端控制和进度通知能力。

**用户需求确认**:
- **模型支持**: DeepSeek、通义千问(Qwen)、智谱GLM、MiniMax
- **部署环境**: 本地Windows
- **并发量**: 单任务处理
- **持久化**: 不需要（内存运行）

## 二、技术架构

### 3.2 Agent角色定义

#### 3.2.1 需求分析师 (Analyst Agent)
- **职责**: 分析用户需求，提取关键信息，生成需求文档
- **系统提示词**: 专注于需求分析、用户故事编写、功能点拆解
- **输出**: 结构化需求文档、功能列表、优先级排序

#### 3.2.2 系统架构师 (Architect Agent)
- **职责**: 设计系统架构、技术选型、模块划分
- **系统提示词**: 专注于架构设计、技术方案、接口定义
- **输出**: 架构设计文档、技术方案、接口规范

#### 3.2.3 代码开发者 (Coder Agent)
- **职责**: 编写代码、实现功能、代码重构
- **系统提示词**: 专注于代码实现、最佳实践、代码质量
- **输出**: 源代码、代码文档、单元测试

#### 3.2.4 测试员 (Tester Agent)
- **职责**: 编写测试用例、执行测试、Bug报告
- **系统提示词**: 专注于测试设计、边界条件、质量保证
- **输出**: 测试用例、测试报告、Bug列表

### 3.4 协议适配层设计

国产大模型大多兼容OpenAI API格式，采用统一基类+差异化处理的方式：

```python
from abc import ABC, abstractmethod
from typing import List, Dict, AsyncIterator
from dataclasses import dataclass
from datetime import datetime

@dataclass
class Message:
    role: str
    content: str
    metadata: Dict = None
    timestamp: datetime = None

class BaseProtocol(ABC):
    def __init__(self, base_url: str, api_key: str, default_model: str):
        self.base_url = base_url.rstrip('/')
        self.api_key = api_key
        self.default_model = default_model
    
    @abstractmethod
    async def chat(self, messages: List[Message], stream: bool = True) -> AsyncIterator[str]:
        pass
    
    @abstractmethod
    def format_messages(self, messages: List[Message]) -> List[Dict]:
        pass
    
    def get_headers(self) -> Dict[str, str]:
        return {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json"
        }

class DeepSeekProtocol(BaseProtocol):
    def __init__(self, api_key: str):
        super().__init__(
            base_url="https://api.deepseek.com",
            api_key=api_key,
            default_model="deepseek-chat"
        )
    
    async def chat(self, messages: List[Message], stream: bool = True) -> AsyncIterator[str]:
        pass

class QwenProtocol(BaseProtocol):
    def __init__(self, api_key: str):
        super().__init__(
            base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
            api_key=api_key,
            default_model="qwen-plus"
        )

class GLMProtocol(BaseProtocol):
    def __init__(self, api_key: str):
        super().__init__(
            base_url="https://open.bigmodel.cn/api/paas/v4",
            api_key=api_key,
            default_model="glm-4"
        )

class MiniMaxProtocol(BaseProtocol):
    def __init__(self, api_key: str, group_id: str):
        super().__init__(
            base_url="https://api.minimax.chat/v1",
            api_key=api_key,
            default_model="abab6.5-chat"
        )
        self.group_id = group_id
```

### 6.2 消息流设计
```python
@dataclass
class Message:
    role: str           # system/user/assistant
```
