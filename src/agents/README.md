# agents - Agent实现

本目录包含具体Agent角色的实现。

## 文件说明

| 文件 | 角色 | 职责 |
|------|------|------|
| `base_agent.py` | 基类 | Agent抽象基类，定义通用接口 |
| `analyst.py` | 需求分析师 | 分析用户需求，输出需求文档 |
| `architect.py` | 系统架构师 | 设计系统架构，输出技术方案 |
| `coder.py` | 代码开发者 | 编写代码，输出源代码 |
| `tester.py` | 测试员 | 编写测试用例，输出测试报告 |

## Agent角色定义

### 需求分析师 (Analyst)
- **输入**: 用户原始需求
- **输出**: 结构化需求文档
- **系统提示词**: 专注于需求分析、用户故事编写、功能点拆解

### 系统架构师 (Architect)
- **输入**: 需求文档
- **输出**: 架构设计文档
- **系统提示词**: 专注于架构设计、技术选型、接口定义

### 代码开发者 (Coder)
- **输入**: 架构设计文档
- **输出**: 源代码
- **系统提示词**: 专注于代码实现、最佳实践、代码质量

### 测试员 (Tester)
- **输入**: 源代码
- **输出**: 测试报告
- **系统提示词**: 专注于测试设计、边界条件、质量保证

## 开发顺序（MVP）

1. `base_agent.py` - Agent基类
2. `analyst.py` - 需求分析师（优先）
3. `coder.py` - 代码开发者（优先）
4. `architect.py` - 系统架构师（后续）
5. `tester.py` - 测试员（后续）
